# iCloud Browser 用户操作说明书

> 欢迎使用 **iCloud Browser** —— 专为 macOS 打造的极速本地相册浏览器。本软件致力于解决系统自带预览工具对海量 HEIC 和 MOV 文件加载慢、卡顿的问题，同时为您带来原汁原味的 Mac 原生操作体验。

---

## 🌟 核心特性 (卖点)

- **⚡️ 极速加载原生缩略图**：底层采用独家研发的 `macos-thumbnail` 引擎，直接调用 macOS `ImageIO` 和 `AVFoundation` 框架。无需等待漫长的格式转换，上万张 HEIC 图像和 H.265 (HEVC) 原生视频缩略图瞬间呈现。
- **💻 原汁原味的 macOS 体验**：
  - **原生外观**：采用全新的透明毛玻璃（Vibrancy）侧边栏，沉浸式悬浮工具栏，完美融入系统。
  - **触手可及的操作**：支持空格键（Space）全局一键 QuickLook 预览。
  - **优雅的多选体验**：完美还原 macOS 的框选和点选逻辑（Command / Shift），随心所欲挑选多张照片。
  - **无缝批量导出**：可以直接拖拽或通过系统右键菜单，一键将数十张甚至上百张照片和视频原生复制到 Finder 或任意文件夹中。
- **📅 优雅的时间线流**：如同 iPhone 相册般顺滑，所有照片自动按拍摄日期归置。向下滚动鼠标，时间线无限加载，回忆自然流淌。
- **🔒 100% 离线隐私安全**：所有照片解析、渲染及数据库存储完全在本地完成，不仅速度极快，且绝不会上传哪怕一字节数据到云端，保护您的绝对隐私。

## 🚀 步骤 1：首次配置与目录加载

1. **打开应用**：双击启动 iCloud Browser，应用会呈现极简的透明主界面。
2. **选择 iCloud 相册目录**：
   - 点击右上角的 **【Select iCloud Photos Directory】** 按钮。
   - 在弹出的原生窗口中，选择您存放 iCloud 备份照片的文件夹（或任何包含海量图片的本地文件夹）。
3. **极速扫描与索引**：
   - 应用会在后台极速扫描您所有的相片和视频（支持 `.heic`, `.mov`, `.jpg`, `.mp4` 等主流格式）。
   - 扫描过程由于基于内部纯原生驱动，上千张照片仅需极短时间即可完成入库。
   - 扫描完成后，**所有照片将自动以“时间线”模式呈现在您的眼前**。

---

## 💡 步骤 2：日常使用与高效操作

### 🖼 时间线浏览 (Timeline View)
- 启动并加载完目录后，直接向下滚动鼠标滚轮。
- 照片已按日自动分组并附带**自动吸顶的日期标题**（例如 `2023-10-24`），让您在时间长河中清晰定位。
- 通过底部的动态加载机制，应用能在一瞬间浏览数万张照片而不会产生任何内存卡顿。

### ⌨️ 键盘与快捷浏览 (QuickLook)
为了提供最丝滑的审核体验，请多尝试使用键盘：
- **单击**：选中任意一张照片（照片会呈现细腻的下钻动效与蓝色光晕，右下角带有 ☑️ 标记）。
- **按下【空格键 Space】**：瞬间以原比例弹出高清大图（或原生播放 `.MOV` 视频）。如果是多选状态，会自动预览所选范围内的第一张图片。
- 双击图片，同样可以直接呼出全屏预览。
- 在预览模式下，按下 **【左 / 右方向键】**：直接切换上一张/下一张图片，审核视频素材极为高效。
- **再次按下【空格键】或【Esc】**：退出全屏预览。

### 🖱️ 原生多选与批量操作 (Multi-select & Batch)
为了方便整理和剪辑，我们深度还原了原生系统的文件选择与传输逻辑：
1. **精准加选**：按住键盘上的 **【Command (Cmd) ⌘】** 键，随心所欲地点击拾取分散在不同日期的多张照片。
2. **范围连选**：先点击一张照片，然后按住 **【Shift ⇧】** 键点击另一张照片，两者之间的所有回忆会被瞬间打包选中。

选好之后怎么做？
- **一步拖拽**：按住选中图片中的任意一张不要松手，直接拖拽到桌面或任何文件夹中，所有选中的文件将被悉数提取！
- **原生批量拷贝**：右键点击选中的集群，选择 **“Copy X Images (复制 X 张图片)”**。这不仅仅是复制像素——它将真实文件路径通过底层接口发送给了 macOS Finder。此时切换到桌面，按下 `Cmd + V`，所有文件原封不动就地粘贴！

### 🏷️ 侧边栏精准过滤
虽然主界面是全局时间线，但左侧依然保留了层级化日期过滤：
- 点击 **年份**（如 2023） -> **月份** -> **日期**：主界面将只显示当日拍摄的照片，方便精细筛选。
- 点击侧边栏最上方的 **【🖼️ All Photos (所有照片)】**：即可立即回到全局无限滚动的岁月时间线。



## 🛠️ 常见问题 (FAQ)

**Q：为什么刚扫描完时部分缩略图空白了一下才出来？**
A：我们的应用采用了“懒加载 + 实时即时渲染”技术。只有当照片滚动进您的屏幕视野时，底层 C++ 引擎才会向 macOS 申请内核资源生成缩略图。这种方式极大地控制了您的硬件内存使用，确保即使您的库里有十万张照片，应用也同样轻快。

**Q：我可以同时管理多个文件夹吗？**
A：当然可以！只需再次点击顶部的【Select iCloud Photos Directory】，选择新的目录，应用会立即切换并扫描新的照片库。

**Q：支持哪些格式的文件？**
A：除了大众的 `jpg/png` 外，本软件的杀手级功能是原生且完美的渲染苹果生态格式：不仅完全支持高画质 `HEIC` 图象，更能直接在瀑布流中提供原生 `H.265(HEVC) MOV` 硬件解码的视频预览体验。
